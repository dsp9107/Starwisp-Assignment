# Starwisp Assignment

# Setup

Step 1

```npm install```

Step 2

Create a .env file in the root directory with these contents

```
PORT=9107
JWT_SECRET='secret-key'
DB_USER=<username to MySQL Server>
DB_PASS=<password to MySQL Server>
DB_HOST=localhost
DB_DATABASE=starwisp
DB_ENV=testing
```

replace `<...>` with your credentials
